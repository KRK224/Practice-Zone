2-2-3. HTML Text Formatting

b tag: Bold의 약자로, 텍스트를 굵게 표시한다. (inline level tag)
strong tag: b tag와 비슷하게 텍스트를 굵게 표시하지만, 의미적으로 더 중요함을 의미. (inline level tag)

i tag: italic의 약자로, 텍스트를 이태릭으로 표시
em tag: emphasize의 약자로, 텍스트를 이태릭으로 표시하는 i tag와 동일하지만, 의미적으로 중요함을 의미.

small tag: 텍스트를 작게 표시.

sup tag: super subscript의 약자로, 윗 첨자를 넣을 때 사용. sub tag: subscript의 약자로, 아랫 첨자를 넣을 때 사용.

오늘은 4/18일 토요일 일요일입니다.
del tag: delete의 약자로, 텍스트를 삭제했다는 의미로 취소선 삽입
ins tag: insert의 약자로, 텍스트를 삽입, 밑줄 표시.
mark tag: 형광펜 강조할 때 사용.

2-2-4. HTML Quotation/Comment

q tag와 blockquote tag.

q tag: quote의 약자이며, 짧은 문장을 인용할 때 쓰이는 태그. 브라우저에서는 "로 표시
blockquote tag는 긴 문장을 인용할 때 쓰이는 태그, 브라우저에서는 들여쓰기로 표시된다.
HTML 요소(HTML 엘리먼트, HTML element)는 HTML 문서나 웹 페이지를 이루는 개별적인 요소를 의미하며, 문서 객체 모델(DOM)으로 파싱된다. HTML은 트리나 HTML 노드(텍스트 노드 등)로 구성된다. 각 노드는 HTML 속성을 지정할 수 있다. 노드들은 기타 노드와 텍스트를 포함한 콘텐츠도 소유할 수 있다.

Link State

Html link(a tag)는 4가지 상태로 정의할 수 있다.
link : 한번도 방문하지 않은 상태 (디폴트); 파란색
visited: 한번이라도 방문한 상태; 보라색
active: 링크를 마우스로 누른 상태; 빨간색
hover: 링크 위에 마우스를 올려놓은 상태; CSS3로 사용자가 설정 가능
CSS3를 사용하여 기본적인 밑줄 및 색상 변경 가능.

2-2-6. HTML Image

<img> tag: image의 약어이며, End Tag가 없는 Empty tag.

사용법

<img src = "url" alt = "대체 텍스트">
src: source의 약자로, 이미지가 있는 URL 주소를 입력.
alt: alternative의 약자로, URL 주소가 틀리거나 이미지를 불러올 수 없을 때 대체로 보여줄 텍스트.

절대 경로 vs 상대 경로

  • 절대 경로: 웹페이지나 파일이 가지는 고유 경로, 최상위 디렉토리를 포함한 주소
  • 상대 경로: 현재 위치를 기준으로 해당 파일이 어디 있는지 표현
    • / : 최상위 디렉토리로 이동한다.
    • ./ : 현재 위치를 의미
    • ../ : 바로 위의 상위 디렉토리로 이동.

width속성, height속성

사용법

<img src="url" alt="대체 텍스트" width="숫자" height="숫자">

2-2-7 HTML List


List는 크게 3가지 종류가 있다.

  1. 순서가 없는 리스트(unordered list)
  2. 순서가 있는 리스트(ordered list)
  3. 정의 리스트(definition list)

순서가 없는 리스트(unordered list)

사용법

                        <ul>
                            <li>항목 1</li>
                            <li>항목 2</li>
                            <li>항목 3</li>
                        </ul>
ul: unordered list의 약자이며, 순서가 없는 리스트를 정의한다. li: ul /ul 싸여진 항목은 li tag를 사용하여 항목을 표시한다.

예시

  • HTML5
  • CSS
  • Java Script

CSS3을 이용하면 검음색 점을 다음과 같은 모양으로 바꿀 수 있다.
style="list-style-type"을 이용한다.

  • disc: 디폴트 값으로 검은색 점이다.
  • circle: 원
  • square: 사각형
  • none: 아무것도 표시되지 않는다.

사용법

                        <ul style="list-style-type:square">
                            <li>항목 1</li>
                            <li>항목 2</li>
                            <li>항목 3</li>
                        </ul>

예시

  • HTML5
  • CSS3
  • Java Script

순서가 있는 리스트

사용법

                    <ol>
                        <li>항목 1</li>
                        <li>항목 2</li>
                    </ol>
ol: ordered list의 약자이며, 순서가 있는 리스트를 정의한다.
type 속성을 이용하면 여러가지 모양으로 바꿀 수 있다.
  • 1: 디폴트 값으로 아라비아 숫자이다.
  • A: 대문자 알파벳
  • a: 소문자 알파벳
  • I: 대문자 로마숫자
  • i: 소문자 로마숫자

사용 예시

  2. 항목 2
  3. 항목 3
  1. 항목 1
  2. 항목 2
  3. 항목 3

정의 리스트 (definition list)

용어와 용어에 대해 정의를 나열하는 List.

사용법

                        <dl>
                        <dt>용어 1</dt>
                        <dd>용어 1에 대한 정의 </dd>
                        <dt>용어 2</dt>
                        <dd>용어 2에 대한 정의 </dd>
                        </dl>
dl : definition list의 약어이며, 정의 리스트를 정의.
dt : 정의할 용어를 입력.
dd : 용어에 대한 정의.
