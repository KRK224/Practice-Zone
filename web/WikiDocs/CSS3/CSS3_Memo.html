03장-2 CSS3를 HTML5에 적용하기

CSS3 HTML5에 적용하는 방법은 크게 3가지가 있다
  • 인라인(inline) 스타일 시트 : HTML문서 내의 각 Tag 요소에 직접 스타일 시트를 정의한다.
  • 내부(internal) 스타일 시트 : HTML문서의 <head> 부분에 스타일 시트를 정의한다.
  • 외부(external) 스타일 시트 : 스타일 시트를 별도의 파일(확장자 .css로 저장하여 다른 HTML문서에서도 같은 스타일 시트를 사용할 수 있도록 합니다.
    • 더 이상 자세한 설명은 생략한다.

    외부(external) 스타일 시트

    각각의 HTML문서의 <head> Tag 내에서 <link> Tag를 이용하여 선언.
    <link rel="stylesheet" type="text/css" href="css 파일명">

03장-3 CSS 적용 우선 순위

  1. importance : 속성 값 뒤에 !important로 설정하면 다른 것들을 고려하지 않고 가장 우선적으로 고려한다.
  2. specificity : 모든 설정이 같을 경우, 가장 나중에 선언한 것이 우선되어 적용됩니다.
    1. 인라인(inline) 스타일 시트
    2. 외부(external) 또는 내부(internal) 스타일 시트
      • id 선택자 > class 선택자 > Tag 선택자
    3. 브라우저 기본 값

03장-8 위치(Position) 다루기

위치(position) 지정 속성의 종류는 크게 4가지가 있다.

  • static : 디폴트 값이며 코드 흐름에서 자신의 위치
  • relative : 원래 위치에서 상대적인 위치 지정
  • absolute : 자신을 감싸고 있는 Tag(static일 경우 적용 안됨 body로 적용됨)에서 상대적인 위치 지정
  • fixed : 전체 보여지는 화면에서 고정된 위치

relative

  • 원래 위치에 대해 상대적으로 위치시킨다.
  • top, bottom, left, right 속성을 지정하여 원래 위치로부터 변경한다
  • 다른 요소들에게 영향을 미치지 않는다

예시

position: relative

relative으로 위치 지정

absolute

  1. 바로 자신을 감싸는 Tag에 대해 상대적으로 위치를 설정한다
  2. 자신을 감싸는 Tag가 없을 경우 <body> Tag를 기준으로 한다
  3. 단, 자신을 감싸는 Tag가 static 위치 속성을 가지면 해당하지 않는다

예시

position: absolute

relative으로 위치 지정

absoulte으로 위치 지정

3번 예시

자신을 감싸는 Tag가 static 속성을 가지고 있어 <body> Tag를 기준으로 한다

position: absolute_static

static으로 위치 지정

absoulte으로 위치 지정

fixed

  • 페이지가 스크롤되더라도 항상 같은 영역에 머무른다.
  • top, bottom, left, right 속성을 이용하여 위치 설정

예시

position: fixed

fixed으로 위치 지정
