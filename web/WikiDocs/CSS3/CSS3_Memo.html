03장-2 CSS3를 HTML5에 적용하기

CSS3 HTML5에 적용하는 방법은 크게 3가지가 있다
  • 인라인(inline) 스타일 시트 : HTML문서 내의 각 Tag 요소에 직접 스타일 시트를 정의한다.
  • 내부(internal) 스타일 시트 : HTML문서의 <head> 부분에 스타일 시트를 정의한다.
  • 외부(external) 스타일 시트 : 스타일 시트를 별도의 파일(확장자 .css로 저장하여 다른 HTML문서에서도 같은 스타일 시트를 사용할 수 있도록 합니다.
    • 더 이상 자세한 설명은 생략한다.

    외부(external) 스타일 시트

    각각의 HTML문서의 <head> Tag 내에서 <link> Tag를 이용하여 선언.
    <link rel="stylesheet" type="text/css" href="css 파일명">

03장-3 CSS 적용 우선 순위

  1. importance : 속성 값 뒤에 !important로 설정하면 다른 것들을 고려하지 않고 가장 우선적으로 고려한다.
  2. specificity : 모든 설정이 같을 경우, 가장 나중에 선언한 것이 우선되어 적용됩니다.
    1. 인라인(inline) 스타일 시트
    2. 외부(external) 또는 내부(internal) 스타일 시트
      • id 선택자 > class 선택자 > Tag 선택자
    3. 브라우저 기본 값
